## 世界级WebGL渲染引擎技能范围

### 第一层：生存底线（必须具备，否则无法竞争）

| 领域 | 核心技能 |
| :--- | :--- |
| **API抽象** | WebGL 1.0/2.0 统一封装，能力探测，优雅降级 |
| **基础渲染** | 几何体/纹理/材质基础管线，顶点/片元着色器 |
| **光照系统** | 方向光/点光源/聚光源，Phong/Blinn-Phong，阴影映射 |
| **场景图** | 树形节点结构，局部/世界变换矩阵 |
| **资源管理** | 异步加载，基础缓存，常用格式(glTF/OBJ) |
| **相机系统** | 透视/正交相机，轨道控制基础 |

### 第二层：工业标准（当今优秀引擎的标配）

| 领域 | 核心技能 |
| :--- | :--- |
| **PBR渲染** | 金属粗糙度流程，IBL环境光照明 |
| **高级阴影** | 级联阴影贴图(CSM)，PCF/PCSS软阴影 |
| **后处理管线** | 泛光、SSAO、色调映射、抗锯齿(FXAA/SMAA) |
| **高级材质** | 多层材质，遮罩/混合，纹理变换，UV动画 |
| **LOD系统** | 距离驱动精度切换，几何体简化 |
| **合批优化** | 静态合批，动态实例化，材质排序 |
| **遮挡剔除** | 视锥剔除 + GPU遮挡查询 |
| **压缩支持** | Draco(几何)，Basis/KTX2(纹理)，Meshopt |
| **骨骼动画** | 蒙皮矩阵，动画混合，关键帧插值 |
| **调试工具** | 场景检视器，性能面板，Draw Call统计 |

### 第三层：卓越突破（世界级引擎的差异化优势）

| 领域 | 核心技能 |
| :--- | :--- |
| **WebGPU原生** | 完整WebGPU后端，计算着色器第一支持 |
| **GPU驱动渲染** | 剔除/蒙皮/粒子在GPU完成，CPU几乎零负担 |
| **虚拟几何体** | 类Nanite技术，影视级模型直接导入，自动流式LOD |
| **虚拟纹理** | 无限分辨率纹理，动态贴花系统 |
| **高级GI** | 实时全局光照，光线追踪(有限场景)，DDGI |
| **TAA/时域抗锯齿** | 稳定抗锯齿，运动矢量支持 |
| **多线程架构** | 无锁资源加载，并行场景图，Worker池 |
| **自适应渲染** | 动态画质调整，设备性能探测，帧率锁定 |
| **高级动画** | 混合树，动画状态机，根部运动，变形目标阵列 |
| **物理集成** | 碰撞体同步，软体/布料模拟接口 |

### 第四层：未来前沿（定义下一代引擎的边界）

| 领域 | 核心技能 |
| :--- | :--- |
| **Mesh Shaders** | GPU端几何体生成，程序化细分曲面 |
| **神经渲染** | 实时超分辨率，AI补帧，深度学习抗锯齿(DLAA) |
| **场景语义理解** | AI辅助材质分类，自动LOD生成，智能合批建议 |
| **跨引擎协作** | 原生glTF扩展，实时联动(Blender/Unity/Unreal) |
| **WebXR深度** | 注视点渲染，变焦深度，立体图层合成 |
| **云流式渲染** | 大场景分块流式，远端渲染+本地合成 |
| **非欧几里得渲染** | 特殊空间变换，递归/反射场景支持 |
