# Simplora WebGL

A lightweight WebGL rendering engine inspired by Simplora, built from scratch without external dependencies.

## 🎯 项目理念

Simplora WebGL 的核心理念是 **极致简单、极致高效**，致力于为开发者提供最简单、最直观的 3D WebGL 开发体验。


### 核心原则

**🚀 极度简单**

- **对接简单**：零依赖，一行代码即可集成，无需复杂的配置和构建流程
- **使用简单**：直观的 API 设计，3 行代码即可创建 3D 场景
- **理解简单**：清晰的代码结构，完善的文档和示例，新手也能快速上手

**⚡ 极度高效**

- **性能高效**：优化的渲染管线，最小化资源占用，最大化渲染性能
- **开发高效**：简洁的 API，减少样板代码，提升开发效率
- **运行高效**：轻量级设计，快速加载，流畅运行


### 设计哲学

- **少即是多**：只提供最核心、最常用的功能，避免过度设计
- **约定优于配置**：合理的默认设置，减少用户的配置负担
- **渐进式增强**：从简单开始，按需扩展，不强制用户学习复杂特性
- **性能优先**：在设计和实现的每个环节都优先考虑性能影响


## Features

- WebGL rendering engine
- Basic 3D primitives (Box, Sphere)
- Material system with color and opacity support
- Camera with perspective projection
- Matrix transformations (translation, rotation, scale)
- Wireframe rendering mode
- Scene graph management


### 目标用户

- 快速原型开发的开发者
- 需要 3D 可视化的数据应用
- 教育和学习 WebGL 的初学者
- 追求性能和简洁性的项目团队
